# Issue Tracker

This repo tracks issues in GitHub Issues for `ssannnn/legalfit`.

Use the GitHub issue tracker for PRDs, implementation issues, triage output and agent-ready tasks.

## Repository

- Owner: `ssannnn`
- Repo: `legalfit`
- Remote: `git@github.com:ssannnn/legalfit.git`

## Workflow

- Create issues with `gh issue create` when publishing implementation tasks.
- Read issues with `gh issue list` and `gh issue view`.
- Apply triage labels using the vocabulary in `docs/agents/triage-labels.md`.
- Prefer linking back to source docs in this repo, especially PRDs under `docs/prd/`.

## Notes

- If the GitHub remote changes, update this file and `AGENTS.md`.
- If GitHub Issues are disabled for the repo, switch this file to the actual issue tracker before using `to-issues`, `triage` or `to-prd`.
